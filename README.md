# FAT32 파일 시스템 비할당영역 카빙 도구



활성화된 FAT32 파일 시스템의 비할당영역에 존재하는 파일들의 포맷을 확인 및 출력해주는 도구



## Usage

``` 
python Fat32Carving.py

Enter Drive name (ex, C, D)
```



## Supporting File Format

```
zip, docx, pptx, ppt, xlsx, pdf, jpeg, png, hwp, winzip, PE File
```
